# Smart Home IoT Control System

A modern Smart Home IoT mobile application UI/UX designed in Figma. The application provides a centralized dashboard to monitor and control smart home devices with a clean, intuitive, and user-friendly interface.

## Features
- 🏠 Smart Home Dashboard
- 🌡️ Weather and Temperature Display
- 📍 Current Location Widget
- 💡 Room-wise Device Management
- ❄️ Air Conditioner Control Screen
- 🌿 Eco Mode Support
- 💧 Humidity & Temperature Monitoring
- 🎨 Modern Orange & White UI Design
- 📱 iOS-inspired Mobile Interface
